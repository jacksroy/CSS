Adjacent Sibling Selectors

This Selectors tell that the next tag of that current tag should has this style

Now p tag has Adjacent Sibling is this h3 tag

Now this p tag is not adjacent sibling to p tag above h3 tag because this is not direct adjacent to the p tag 

        Syntax:

        current-tag + Adjacent-Sibling-tag 
        
        {
            property1:value1;
            property2:value2;

        }

        Example:

        div + p
        
        {
            color:green;
            font-size:36px;
        }
