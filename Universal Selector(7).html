This is Universal Selectors

Universal Selectors Select all elements in the document without leaving any element or tag in the document

It gives same values to all the document

Universal Selector use * to define the selector is universal Selector

        Syntax:

        *{
            property1:value1;
            property2:value2;
            ......

        }

        Example:

        *
        {
            color:white;
            background-color:black;


        }
