Id Based Selectors

Paragraph 1 with id name of id1

Paragraph 2 with id name of id2

Paragraph 3 with id name of id3

        Syntax:

        #id_name {

            property1:value1;
            property2:value2;

        }

        Example:

        #id1{

            color:red;
            font-size:10%;

        }

        #id2
        
        {

            background-color:green;
        }
