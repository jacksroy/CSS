we are applying some thing using some attribute selector 

            selector [attribute] {
                property :value;
            }

we are doing now the attribute with name 

        selector [attribute="value"] {
            property :value;
        }

we are doing now the attribute with name with ^ means the value can be exactly match this it can be a part of that value 

         selector [attribute^="value"] {
            property :value;
        }

we are doing now the attribute with name with $ means the value is present at beginning 

         selector [attribute$="value"] {
            property :value;
        }

we are doing now the attribute with name with * means the value is present at ending 

         selector [attribute*="value"] {
            property :value;
        }
